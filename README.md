# Offline-resource-library
OneDrive文件网页版

无法登录OneDrive客户端的用户可在此下载资源！

网页版优点：不用登录，一键下载。

网页版缺点：资源更新较慢，大于100MB的文件无法上传，速度堪忧qwq

解压密码：询问管理员/查看OneDrive客户端
